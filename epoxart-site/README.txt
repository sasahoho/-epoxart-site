ÉPOX'ART — SITE WEB

1. Ouvrir index.html dans un navigateur pour prévisualiser.
2. Pour publier sur Vercel : importer le dossier complet dans un dépôt GitHub, puis importer le dépôt dans Vercel.
3. Le formulaire de contact est volontairement laissé en mode démonstration. Il faudra ajouter l'adresse courriel de l'entreprise ou connecter Formspree/Resend.
4. Remplacer les aperçus de finitions par de vraies photos dès les premiers chantiers.
5. Connecter ensuite le domaine acheté dans Settings > Domains sur Vercel.

Fichiers :
- index.html : contenu du site
- styles.css : design
- script.js : menu, animations et calculateur
- assets/ : logo et visuels de marque
